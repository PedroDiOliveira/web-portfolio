## Instalação

Siga os passos abaixo para rodar o projeto localmente:

1. Clone o repositório:

    ```bash
    git clone https://github.com/PedroDiOliveira/web-portfolio.git
    ```

2. Instale as dependências:

    ```bash
    cd web-portfolio
    npm install
    ```

3. Inicie o servidor de desenvolvimento:

    ```bash
    npm run dev
    ```
